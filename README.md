#Odin Recipes

##Introduction
This project is part of [The Odin Project curriculum](https://www.theodinproject.com/lessons/foundations-recipes#project-submissions-and-reasonable-expectations). It focuses on building a simple recipe website using **HTML fundamentals.** The goal is to create a multi-page site that links together different recipes, demonstrating your ability to structure content semantically and use relative paths for navigation.

## Project Overview
The site includes:
- A homepage listing available recipes.
- Individual pages for each recipe (e.g., Lasagna, Pizza) with:
  - A description of the dish.
  - Ingredients list.
  - Step-by-step instructions.


### What I Learned
- How to clone GitHub repo via SSH
- How to structure a basic HTML project.
- The importance of semantic tags for accessibility and clarity.
- How to use relative paths for linking pages and assets.
- How to deploy a static site using GitHub Pages.
